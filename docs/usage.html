Wake Processing¶

  1. Import appropriate modules.
from waketracking.dataloaders import foam_ensight_array
from waketracking.waketrackers import track
  1. First load the data using the appropriate data loader (if necessary), as discussed in the previous section.
arraydata = foam_ensight_array(
        outputDir='postProcessing/array.3D',
        prefix='array.3D_U')
  1. Since the input data may be three-dimensional, extract a slice of the data (if necessary).
y,z,u = arraydata.sliceI(0)         # slice by index
y,z,u = arraydata.slice_at(x=1000.) # slice at location
  1. Perform the wake analysis. The track function may be called without a specified method to return a list of available tracking methods.
wake = track(\*args,\*\*kwargs, method='ConstantArea')
